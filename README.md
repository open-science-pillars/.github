# .github

Org-wide community files for Open Science Pillars: the profile README,
five issue templates (bug, feature, new skill, new domain plugin, new
knowledge concept), the PR template with the DCO and review checklist,
GOVERNANCE.md (federated repository authority, interim solo stewardship, and
future provider teams), the Contributor Covenant v2.1, and SECURITY.md. Each
repository declares its roadmap proposal policy in `.osp/governance.yaml`.
